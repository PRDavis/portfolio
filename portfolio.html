Appify Modal


Single-page responsive application built with Knockout.js framework. Developed a full-page map that loads with unique restaurants from McDonough, GA. Application allows the user to view pictures, and reviews from Yelp.com. User can click through to Yelp for more information or click the link to call the restaurant.

Technologies used:

  • HTML5
  • Javascript
  • CSS3
  • Knockout.js
  • Bootstrap
  • Jquery
  • AJAX
  • Google Maps API
  • Yelp API
  • Oauth
